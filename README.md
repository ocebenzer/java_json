# JAVA_JSON v1.0
This library supports minimal JSON parsing features in Java.

### Features
- `JSONValue.parse(String)`: (String -> Object)
- `toString()`: (Object -> String)
- `get()`: to extract value
- `getType()`: returns enum value of **JSONValue**
